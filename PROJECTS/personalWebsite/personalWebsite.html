Create a Personal Website | Andrew Cappelli

This guide aims to provide a clear and insightful overview of creating and hosting your own personal website. A condition of this tutorial is the use of VSC or visual studio code. Adittionally the use of a domain name and cloudflare is optional.

Rquirements

  • GitHub Account
  • Cloudflare Account
  • Domain Registrar
  • IDE of your choice
  • Git

Step 1: Create a GitHub Repository

  1. Log in to your GitHub account.
  2. Click the "+" icon in the top right corner of the page and select "New repository."
  3. Enter a name for your repository, such as "personal-website."
  4. Select the readme box to add a ReadMe to your repository.
  5. Make sure to set your repository to public.
  6. Click "Create repository."

Step 2: Install git

  1. Once you've installed run Git Bash
  2. Run the following commands:
    • git config --global user.name "Your Name"
    • git config --global user.email "Your Email"
    • Close Git Bash

Step 3: Clone your repository

  1. Open Your IDE (were using VSC)
  2. Slect the Accounts icon in the bottom left
    and sign in to your github account
  3. Press Ctlr + Shift + E or select the page icon
  4. Click the "Clone Repository" button
    • Either Enter the URL of your repository
      Select your repository in the drop down
    • Choose a location to save your repository
    • Click "Clone"

Step 4: Create your website

  1. Right click on your repository in the explorer
  2. Select "New File"
  3. Enter "index.html" as the file name
  4. Enter the following code into your file: 
                    
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>My Personal Website</title>
                    </head>
                    <body>
                        <h1>Welcome to My Personal Website!</h1>
                    </body>
                    </html>

Step 5: Push your changes

  1. press Ctrl + Shift + G or select Sorce Control
  2. Enter a commit message
  3. Press the Plus Icon in the changes list
  4. Press the commit button to commit your changes
  5. Finally Sync Chnages

Step 6: Setup Github Pages

  1. Go to your repository on GitHub
  2. Click on the "Settings" tab
  3. Select "GitHub Pages" in the left side actions list called "Code and automation"
  4. Under "Source" select "main" or "master" branch
  5. Click "Save"
  6. After a few minutes, your website will be live at https://yourusername.github.io/yourrepositoryname

Step 7: Optional - Purchase a Domain Name

  1. Go to a domain registrar such as Namecheap
  2. Search for your desired domain name
  3. Click "Add to Cart" and complete the purchase process
  4. Go to your domain registrar's DNS settings
  5. under name server select custom dns
  6. Enter the nameservers provided by your hosting provider
  7. Wait for the DNS changes to propagate (this can take up to 48 hours)
  8. Your website will now be live at your custom domain name

Step 8: Optional - Setup a Custom Domain with Cloudflare

  1. Go to Cloudflare and create an account
  2. Click on "Add a site" and enter your domain name
  3. Click "Add site"
  4. Cloudflare will scan your DNS records
  5. Click "Continue"
  6. Choose a plan (Free is fine for most users)
  7. Click "Continue"
  8. Cloudflare will provide you with two nameservers
  9. Go to your domain registrar and update your nameservers to the ones provided by Cloudflare
  10. Go back to Cloudflare and click "Done"

Step 9: Optional - Finishing Your Custom Domain Name

  1. Go to Cloudflare and click on the "DNS" tab
  2. Click "Add record"
  3. Create an A record for all of the following IP addresses
  4. make sure type is A Record name is "@" and ip is the one of the following github pages IP's
    • 185.199.111.153
    • 185.199.110.153
    • 185.199.109.153
    • 185.199.108.153
  5. Click "Save"
  6. Click "Add record"
  7. make sure type is CNAM Name required is "www" is your custome domain
  8. Click "Save"
  9. Go Back into the Github Pages settings find "Custom domain"
  10. Enter your custom domain name
  11. Click "Save"
  12. Note that it will not work yet
  13. Wait for the DNS changes to propagate (this can take up to 48 hours)
